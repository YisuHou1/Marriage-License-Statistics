# Marriages in Toronto before and after the pandemic

## Overview

This repo analyzes the number of marriage licenses before and after the Covid pandemic.


## File Structure

The repo is structured as:

-   `data` contains all data used for analysis, including raw, simulated, and cleaned data.
-   `other` contains the planning sketch for the analysis.
-   `documents` contains the files used to generate the paper, including the Quarto document and reference bibliography file, as well as the PDF of the paper. 
-   `scripts` contains the R scripts used to simulate, download and clean data.
